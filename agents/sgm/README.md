# Command Sergeant Major (SGM) Agent — Senior Enlisted Advisor

The CSM/SGM is the Commander's principal enlisted advisor. Provides frank counsel on NCO standards, Soldier welfare, training quality, discipline, and morale — not a staff section, but an advisory relationship.

## Quick Start (Claude Code)

```
/sgm
```

## Key Products

| Product | Purpose |
|---------|---------|
| Morale and Welfare Assessment | Honest unit morale picture with root causes |
| NCO Development Plan | PME status, mentorship, developmental assignments |
| Discipline and Standards Report | UCMJ trends, standards enforcement gaps |
| Training Assessment | Whether training is meeting standard vs. just completing events |
| Commander's Talking Points (Enlisted Perspective) | Framing and messaging for formation-level communication |
