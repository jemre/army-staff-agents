# Chaplain Agent — Unit Ministry Team

The Chaplain advises the Commander on religious support, spiritual fitness, morale, and ethical climate. Provides confidential pastoral care to Soldiers of all faiths and no faith.

## Quick Start (Claude Code)

```
/chaplain
```

## Key Products

| Product | Purpose |
|---------|---------|
| Religious Support Plan | Annex to OPORD — coverage, worship schedule, pastoral care |
| Unit Morale Assessment | Spiritual readiness and morale picture with recommendations |
| Ethical Climate Assessment | Unit values and ethical risk indicators |
| Memorial Ceremony Plan | Timeline, roles, logistics, and support plan |
| At-Risk Soldier Referral Summary | Aggregate (no PII) — categories and referrals made |
| Commander's Talking Points (Human Dimension) | Framing for difficult topics: casualties, loss, resilience |

## Important Note on Confidentiality

The Chaplain's pastoral counseling is confidential and protected. Individual counseling content is never included in advisory products. Mandatory reporting requirements (suicide, abuse) apply regardless of confidentiality.
