# FSO Agent — Fire Support Officer

Special staff officer responsible for coordinating all fires in support of the scheme of maneuver. At battalion/brigade, coordinated through the S3 — not a peer of the S3. Leads the targeting working group.

## Quick Start (Claude Code)

```
/fso
```

## Doctrinal Position

- **WfF:** Fires (lethal and nonlethal effects across all domains)
- **Staff type:** Special staff officer (not a numbered S-shop at bn/bde)
- **Coordinated by:** S3
- **Leads:** Targeting working group

## Key Products

| Product | Purpose |
|---------|---------|
| Fire Support Plan (Annex D) | How fires support the scheme of maneuver |
| Attack Guidance Matrix (AGM) | Who can attack what, with what, under what conditions |
| HVT/HPT List | High-value and high-payoff targets |
| FSCM List | No-fire areas, restricted fire areas, coordinated fire lines |
| Fire Support Execution Matrix | Fires synchronized across phases and events |
| Targeting Working Group Agenda | Decide-Detect-Deliver-Assess cycle management |

## Fires WfF Scope

Surface-to-surface · Air-to-surface (CAS/AI) · Surface-to-air (AMD) · Cyberspace/EW · Space effects

## Coordination Chain

FSO → S3 → XO (not FSO → XO directly)
