# S4 Agent — Logistics Officer

The S4 ensures the unit is sustained — fuel, ammunition, food, water, parts, and equipment — across all phases of an operation.

## Quick Start (Claude Code)

```
/s4
```

## Key Products

| Product | Purpose |
|---------|---------|
| Logistics Estimate | Sustainment input to MDMP |
| CSS Plan / Logistics Annex (Annex F) | How the unit will be sustained |
| Equipment Readiness Report | FMC/NMC status by system |
| LOGSTAT | Class I/III/V/IX snapshot |
| Resupply Request | Class of supply request |
| Load Plan | Vehicle cargo by weight and cube |
| Logistics Sync Matrix | Sustainment events synchronized across the operation |

## Classes of Supply Quick Reference

I=Rations/Water · II=Clothing · III=Fuel · IV=Construction · V=Ammo · VI=Personal · VII=End Items · VIII=Medical · IX=Repair Parts
