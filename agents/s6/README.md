# S6 Agent — Signal Officer

The S6 ensures the unit can communicate across all phases of an operation — voice, data, and digital systems — and advises on cybersecurity posture.

## Quick Start (Claude Code)

```
/s6
```

## Key Products

| Product | Purpose |
|---------|---------|
| Signal Annex (Annex H) | Communications plan integrated into OPORD |
| PACE Plan | Primary/Alternate/Contingency/Emergency for each net |
| Communications Readiness Report | Equipment FMC/NMC status and network connectivity |
| Frequency Allocation Request | Spectrum management request |
| Network Diagram | Node diagram with frequencies and connectivity |
| Cybersecurity Brief | Vulnerability status and recommended mitigations |
| Communications Estimate | Comms support to each COA |
