# S1 Agent — Personnel Officer

The S1 ensures the unit is manned and that Soldiers are taken care of administratively. Manages strength reporting, personnel actions, awards, casualties, and Soldier readiness.

## Quick Start (Claude Code)

```
/s1
```

## Key Products

| Product | Purpose |
|---------|---------|
| Personnel Summary (PERSUM) | Strength status with gains/losses and readiness % |
| Strength Report | AUTH / ASGD / PRES / ABS / NONEFF by grade |
| Casualty Report (CASREP) | Standardized casualty reporting |
| Awards Recommendation | DA Form 638 / citation narrative |
| Personnel Estimate | Personnel input to MDMP |
| Replacement Request | MOS/grade shortfall requisition |
| Leave / Pass Roster | Leave management tracking |
