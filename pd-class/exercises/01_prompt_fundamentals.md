# Exercise 01 — Prompt Fundamentals
*Phone-friendly | 3 minutes | Used during Section 3*

---

## Instructions for Students

Open any AI app on your phone (Claude, ChatGPT, Gemini — any works).

**Step 1:** Type your prompt however you naturally would. Use one of these tasks or your own:

> "Help me write a BLUF for a memo recommending a new PT schedule."

> "Write talking points for a safety briefing before a field exercise."

> "Draft an opening paragraph for an award citation for one of my Soldiers."

Send it. Read the output.

---

**Step 2:** Now rewrite your prompt using this structure:

```
Role:    "You are a [role]..."
Task:    "Write a [specific product]..."
Context: "[Key facts the AI needs to know]..."
Format:  "Format it as [structure, length, style]..."
```

Send the new version. Compare the outputs.

---

## Debrief Questions

- What was different about the second output?
- What did you have to add that you didn't expect?
- What would you still need to fix before using the output?

---

## Example Transformation

**Before:**
```
Help me write a BLUF for a memo about PT.
```

**After:**
```
You are a company XO writing a memo for your battalion commander.
Write a BLUF (one sentence, bottom line up front) for a memo
recommending that the company shift PT from 0630 to 0530 start
time. The reason is that the current schedule conflicts with 0700
motor pool PMCS. The BLUF should state the recommendation and
the key reason in plain language.
```

---

## Notes for Instructor

- Don't force participation — make it low-pressure
- Ask 1–2 volunteers to share what they typed and what they got
- The contrast between outputs is more powerful than any explanation
- If someone gets a great first-try output with a weak prompt, use it: "Okay — but would you trust this for something that goes to the commander?"
